# Installation Guide

## Requirements
- Windows 10 or later
- WSL (Windows Subsystem for Linux) installed and configured
- Python 3.8 or later

## Installation Steps
1. Clone the repository
2. Install the required dependencies:
   ```powershell
   pip install -r requirements.txt
   ```
3. Run the application:
   ```powershell
   python -m wsl_shortcut_creator
   ```
